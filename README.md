# Amaris_leatherstore
Amaris Leather Store,Where Craftsmanship Meets Elegance

Amaris Leather is an e-commerce platform designed to provide users with a seamless and modern online shopping experience for high-quality synthetic (fake) leather goods. Developed using an enterprise-level tech stack with Angular, NestJS, PostgreSQL, Prisma, and Tailwind CSS, the system enables secure user authentication, product browsing, cart and wishlist management, and a streamlined checkout process. With admin tools for managing products and orders, the platform ensures efficiency, scalability, and enhanced customer satisfaction while promoting premium faux-leather craftsmanship through digital innovation.

Problem Statement

Traditional physical shopping for premium leather-like goods presents challenges such as limited product availability, restricted store hours, and the inconvenience of traveling to stores. Customers seeking high-quality synthetic (fake) leather items require reliable access to updated product information, easy product comparison, and a smooth purchase experience. Businesses also struggle with manual inventory handling, slow order processing, and limited digital presence.

Amaris Leather addresses these challenges through a fully digital, scalable, and user-friendly platform.

System Overview

1)Angular Frontend (Admin + Customer Dashboard)

2)NestJS Backend (Security + Business Logic)

3)PostgreSQL Database (Products, Orders, Users, Payments)

4)Prisma for type-safe and efficient database access
